Welcome to the Virtual Watershed's modeling platform. If you've been here before you may want to continue along to the modeling dashboard. If not, please read on to learn how to use the Virtual Watershed online modeling system.

This system is built on a number of tools developed by WC-WAVE developers. The main interface is part of this o Virtual Watershed webapp and the modeling service is Moinul Hossain's vwadaptor project. We currently support only two models to be run through the Dashboard. You can also generate gridded data for certain watersheds with the a[href=/modeling/makegrid]{Gridding Tool page}, which was made by Dr. Donna Delparte's lab. Tucker Chapman is currently the lead developer of the project that was initiated by Joel Johansen.

To get started using this system you will need to provide data. Currently we only have instructions ready for iSNOBAL.
